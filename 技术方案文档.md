## 1. 技术栈

| 层 | 技术选型 | 说明 |
|:---|:---|:---|
| **静态站点生成器** | Hugo (Extended, v0.145+) | `go.mod` 中已锁定 `github.com/HugoBlox/kit` 模块 |
| **主题框架** | HugoBlox Kit (Academic CV 模板) | 基于块（Block）的布局系统，所有内容通过 Markdown 驱动 |

| **包管理** | pnpm (Node) | 用于 PostCSS 等前端资源构建，已配置 `package.json` |
| **开发预览** | `hugo server -D` | 本地热重载开发 |

## 3. 语言配置方案

### 3.1 激活双语

在 `languages.yaml` 中取消中文段落的注释，并新增 `content/en/` 和 `content/zh/` 两套内容目录：

```yaml
en:
  locale: en-us
  contentDir: content/en

zh:
  locale: zh-Hans
  contentDir: content/zh
  title: 谢德晨 - 个人学术主页
  params:
    description: 安徽医科大学生物医学工程专业本科生
```

### 3.2 内容组织方式

```
content/
├── en/                      # 英文内容（默认语言）
│   ├── _index.md
│   ├── authors/
│   ├── experience.md
│   ├── events/
│   ├── publications/
│   └── projects/
│
└── zh/                      # 中文内容
    ├── _index.md
    ├── authors/
    ├── experience.md
    ├── events/
    ├── publications/
    └── projects/
```

**翻译范围**：所有公开页面均提供中英双语。`data/authors/me.yaml` 中的 `name` 字段通过 `display` / `alternate` 分别保存中英文名。

### 3.3 导航菜单双语

在 `menus.yaml` 中分别定义 `main`（英文）和 `zh.main`（中文）两套导航：

| 英文 Label | 中文 Label | URL |
|:---|:---|:---|
| Bio | 简介 | `/` |
| Research | 科研 | `/#research` |
| Publications | 论文 | `/#papers` |
| Talks | 报告 | `/#talks` |
| News | 动态 | `/#news` |
| Experience | 经历 | `experience/` |
| Projects | 项目 | `projects/` |

## 7. 技术风险与应对

| 风险 | 影响 | 应对方案 |
|:---|:---|:---|
| Hugo 版本不兼容 | 构建失败 | 锁定 `go.mod` 中模块版本，使用 Hugo Extended v0.145+ |
| 中文内容 CJK 排版 | 渲染异常 | `hugo.yaml` 设置 `hasCJKLanguage: true` |
| 图片路径错误 | 头像不显示 | 使用 `assets/` 目录（Hugo 资源管道管理）而非 `static/` |
| 双语内容不同步 | 导航错乱 | 同时维护 `content/en/` 和 `content/zh/` 两套独立内容 |
