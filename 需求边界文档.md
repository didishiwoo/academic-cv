# 需求边界文档 — 个人简历主页重构

## 一、项目目标
将现有的 Hugo Blox 学术个人主页，重构为一份面向**导师任志华老师**的单页式中文简历主页，让老师能在一次滚动中快速获取你的完整画像。

## 二、目标受众
- **首要读者**：南方医科大学/中国科学院深圳先进技术研究院 任志华老师
- **次要读者**：其他潜在导师或招生委员会成员

## 三、页面结构 — 单页式
所有内容在一个页面内按区块顺序滚动展示，无二级页面。区块顺序如下：

| 区块 | 内容 |
|------|------|
| **1. 顶部身份区** | 姓名、学校、专业、照片、联系方式（邮箱/GitHub/LinkedIn） |
| **2. 教育背景** | GPA 3.57/4、排名 11/119、六级 488、核心课程成绩 |
| **3. 科研经历** | 省级大创项目（四通道时域干涉刺激仪，增强型 Howland 恒流源）+ 神经工程实验室科研经历（EMBC 2025，个性化电导率建模） |
| **4. 荣誉奖项** | 电赛省级二等奖、BME 创新设计国家二等奖、物理实验竞赛国家二等奖、张锡祺一等奖学金 x2、挑战杯校级一等奖 |
| **5. 学术成果** | EMBC 2025 口头报告（哥本哈根） |
| **6. 技能** | 编程与工具、研究方法 |
| **7. 与导师研究的关联** | **核心差异化内容**——展示你对导师 HAPLS 混合匀场系统的理解，以及你正在做的 MRI 兼容性改造工作，表达研究方向契合度 |

## 四、内容要求（信息层次）
- **全部展示**：套磁信中提到的所有关键信息点（排名/GPA、大创项目、三个竞赛、EMBC 口头报告、HAPLS 研究理解、当前 MRI 兼容性工作）全部在首页可见
- 区块 7 为**针对任老师定制**的内容，是该主页区别于通用简历的核心卖点

## 五、语言
- **仅中文**，去掉英文版，降低维护成本

## 六、功能模块取舍
- 去掉：博客/新闻动态、学术报告列表、论文列表页、项目详情页
- 保留精简：代表性论文以列表形式嵌入首页
- 保留：简历下载按钮

## 七、视觉风格
- **简洁学术风**：干净、信息密度高、专业感强
- 可用现有的 Hugo Blox 框架，但大幅简化主题配置和布局
- 配色以中性学术色为主（深蓝/灰色系），避免花哨

## 八、非功能要求
- 保持 Hugo Blox / Hugo 静态站点框架不变（不重新选型）
- 保持现有部署方式（GitHub Pages / Netlify）不变
- 响应式，移动端可正常浏览
